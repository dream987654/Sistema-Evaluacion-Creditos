# Problemática

Durante el análisis realizado se identificó que el proceso de evaluación de créditos presenta retrasos debido al registro manual de información y la falta de centralización de los datos.

## Esto ocasiona:

Mayor tiempo de respuesta.
Dificultad para realizar seguimiento.
Riesgo de errores durante la evaluación.

Con base en ello, el proyecto se centrará exclusivamente en optimizar el proceso de Evaluación Crediticia.
