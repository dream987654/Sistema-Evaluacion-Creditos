# Sistema-Evaluacion-Creditos
Proyecto académico para el desarrollo de un Sistema de Evaluación y Aprobación de Créditos utilizando Scrum Adaptado.

## Descripción

Este proyecto tiene como objetivo diseñar un Sistema de Evaluación y Aprobación de Créditos para una cooperativa de ahorro y crédito.

Como caso de estudio se tomó la Cooperativa Ética y, para efectos académicos, el desarrollo técnico se centra en el Sprint 3 correspondiente al proceso de Evaluación Crediticia.

## Objetivo

Diseñar una solución informática que permita optimizar el proceso de evaluación de solicitudes de crédito mediante la aplicación de buenas prácticas de Ingeniería de Software y Scrum Adaptado.

## Metodología

El proyecto se desarrolla utilizando Scrum Adaptado.

Sprint 2 → Registro de Solicitudes

Sprint 3 → Evaluación Crediticia

Sprint 4 → Aprobación

Sprint 5 → Seguimiento

Sprint 6 → Reportes
