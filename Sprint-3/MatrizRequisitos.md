# Introducción

En este documento se presentan los requisitos funcionales y no funcionales correspondientes al Sprint 3, enfocados en el módulo de Evaluación Crediticia del Sistema de Evaluación y Aprobación de Créditos.
| ID    | Historia de Usuario                                                                     | Prioridad |
| ----- | --------------------------------------------------------------------------------------- | --------- |
| HU-02 | Como evaluador, quiero revisar la información del solicitante para emitir una decisión. | Alta      |

## Requisitos Funcionales
| ID    | Requisito                                      | Prioridad | Historia | Estado   |
| ----- | ---------------------------------------------- | --------- | -------- | -------- |
| RF-01 | Mostrar solicitudes pendientes de evaluación.  | Alta      | HU-02    | Aprobado |
| RF-02 | Consultar información del solicitante.         | Alta      | HU-02    | Aprobado |
| RF-03 | Registrar observaciones durante la evaluación. | Alta      | HU-02    | Aprobado |
| RF-04 | Registrar el resultado de la evaluación.       | Alta      | HU-02    | Aprobado |
| RF-05 | Almacenar la evaluación realizada.             | Alta      | HU-02    | Aprobado |
